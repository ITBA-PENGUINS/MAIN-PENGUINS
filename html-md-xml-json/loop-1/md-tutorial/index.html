Що таке MD (markdown) ?

Це полегшена мова розмітки даних, яку створено для полегшення читабельності та зручності публікації з подальшим перетворенням її на XHTML або HTML. Чимало ідей для мови позичено з існуючих домовленостей у розмітці тексту в електронних листах. Першу реалізацію Markdown написано Джоном Грубером на Perl, але з часом з'явились багато реалізацій від сторонніх розробників.

Де використовується MD?

Markdown це неофіційний стандарт для багатьох сайтів пов'язаних з програмуванням, наприклад GitHub, Stack Overflow. Також за замовчуванням md підтримують багато додатків для комунікації, наприклад Skype, Slack, Reddit.
